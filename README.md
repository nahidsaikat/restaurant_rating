# Restaurant Rating

## Run the application 
* `git clone https://github.com/nahidsaikat/restaurant_rating.git`
* `cd restaurant_rating`
* `pipenv install`
* `pipenv shell`
* `python manage.py migrate`
* `python manage.py runserver`
* `python manage.py test`    (To run the test)

## Dependencies
* Python 3.10-*
* Django 4.0
* DRF 3.13.1
